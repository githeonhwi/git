# YouTube Thumbnail Copy Generator

유튜브 영상 링크를 받아 **썸네일 문구 이미지 3개**를 만든다.
실행 전 `~/.claude/references/youtube-thumbnail.md` 를 반드시 읽고 시작한다.

## 입력값

- **YouTube 링크** (일부공개/공개 모두 가능): 필수

## 실행 절차

### 1단계: 영상 내용 파악

`WebFetch`로 영상 페이지를 열어 제목·설명·자막(있으면)을 읽는다.
→ 핵심 주제 1줄, 콘텐츠명, 핵심 키워드 3~5개 추출

### 2단계: 웹 리서치

`WebSearch`로 동일 주제 레퍼런스를 수집한다. 타깃·수집 기준은 레퍼런스 파일을 따른다.

### 3단계: 문구 작성

#### 문구 구조

| 위치 | 명칭 | 역할 | 글자 수 제한 |
|------|------|------|-------------|
| 윗줄 | 서브카피 | "왜 봐야 하는가"의 근거 | **띄어쓰기 제외 12자 이하** |
| 아랫줄 | 헤드카피 | 시청자 행동·감정 직접 자극 | **띄어쓰기 제외 10자 이하** |

#### 원칙

- 콘텐츠명을 서브카피 또는 헤드카피 중 최소 하나에 포함한다.
- 헤드카피 3개는 서로 다른 클릭 심리를 자극한다. 유형 분류는 레퍼런스 파일을 따른다.
- 3개의 서브카피 근거 유형을 달리한다.

### 4단계: 이미지 생성

```bash
python ~/.claude/skills/youtube-thumbnail/scripts/make_thumbnail.py \
  --copies "서브카피1|헤드카피1" "서브카피2|헤드카피2" "서브카피3|헤드카피3" \
  --labels "1안설명" "2안설명" "3안설명" \
  --out-dir "./out"
```

환경 준비: `python3 -m pip install Pillow -q`

## 출력 형식

이미지 생성 전 텍스트로 먼저 보여준다:

```
📌 핵심 주제: [한 줄 요약]
🔍 리서치 근거 지표: [수집된 검증 지표]

1번 이미지 ([유형])
  서브카피: [문구]  ([글자수]자)
  헤드카피: [문구]  ([글자수]자)
...
```

## 기본 원칙

- 글자 수는 띄어쓰기를 제외한 실제 글자 수로 계산한다. 초과하면 재작성한다.
- 콘텐츠명은 3개 이미지 모두에 포함한다.
- 시간 기반 문구("10분 요약" 등)는 영상 길이 확인 후에만 사용한다.
